# بحث ودراسة في المكتبة الشاملة (إضافة لتطبيق كلود)

إصدار **١٫٠٫٠** — أداة بحث ودراسة عميقة لتطبيق كلود تتيح الوصول الكامل إلى **المكتبة الشاملة ٤** المُثبَّتة على جهازك (المكتبة الإسلامية).

عشرون أداة بحث وقراءة محلية. كلُّ شيء يحدث على جهازك ولا يُرسَل أيُّ بيانات عبر الشبكة.

---

## ١) ماذا تفعل هذه الإضافة

تتيح لكلود بناء أبحاث موثَّقة بالمراجع داخل مكتبتك الشاملة. تشمل القدرات الأساسية:

- البحث في نصوص آلاف الكتب المُنزَّلة (المتن، الحواشي، التعليقات).
- البحث الصرفي عبر تحليل الجذور (محرِّك «الخليل»)، أو البحث بالحروف البديلة (`*` و`?`).
- تصفية النطاق بالكتب والمؤلفين والتصنيفات وفترات هجرية.
- قراءة كاملة للصفحات والأبواب والفهارس.
- البحث في القرآن الكريم (٦٢٣٦ آية) وجلب الآيات بالرسم الإملائي والعثماني.
- ربط الآيات بتفاسيرها، والأحاديث بمصادرها، عبر فهارس الشاملة الجاهزة.
- صياغة الإحالات بنمط المكتبة الشاملة الافتراضي ليصلح للنشر مباشرة.

## ٢) التثبيت

١) نزِّل ملف الإضافة (بامتداد `.mcpb`) من صفحة الإصدارات على المستودع.

٢) في تطبيق كلود، افتح: **الإعدادات ← الإضافات ← تثبيت إضافة**، ثم اختر الملف. **لا تعتمد على النقر المزدوج** — تربيط الملف في ويندوز ليس موثوقًا، اعتمد على قائمة الإعدادات.

٣) في نافذة الإعداد حقلان اختياريان (في الغالب اتركهما فارغين):

- **مجلد المكتبة الشاملة** — يُكتشف تلقائيًّا. املأه فقط إذا ثبَّت المكتبة في مكان غير معتاد.
- **مسار جافا** — يُستخدم جافا المرفقة مع المكتبة الشاملة افتراضيًّا.

٤) اضغط «تثبيت»، ثم **أغلِق تطبيق كلود إغلاقًا كاملًا** (نقرة يمين على أيقونة شريط النظام، ثم «إنهاء»)، وأعد فتحه.

## ٣) التحقُّق من نجاح التثبيت

في محادثة جديدة جرِّب:

> ابحث في المكتبة الشاملة عن «الكلام» وأخبرني في أيِّ الكتب وردت ومع من.

إذا ظهرت لك نتائج تشمل اسم الكتاب والمؤلف ورقم الصفحة المطبوعة ومقتطفًا من النص — الإضافة تعمل.

## ٤) ما الذي تحتاجه

- نظام ويندوز ١٠ أو ١١ (التوافق مع نظام ماك ممكن لكن لم يُختَبر بعد).
- **المكتبة الشاملة ٤** مُثبَّتة على جهازك مع تنزيل **كتاب واحد على الأقل** من داخل تطبيق المكتبة. (التطبيق يبقى كما هو، والإضافة تقرأ من قاعدة بياناتها قراءةً فقط).
- **تطبيق كلود لسطح المكتب** من الموقع الرسمي.

## ٥) الأدوات العشرون

| الأداة | الوظيفة |
|---|---|
| `shamela_search_pages` | بحث في الصفحات (المتن والحواشي) مع نطاق وخيارات صرف وبدلاء. |
| `shamela_search_titles` | بحث في عناوين الفصول والأبواب. |
| `shamela_search_books` | بحث في فهرس ٨٥٠٠+ كتاب. يعمل قبل تنزيل أي كتاب. |
| `shamela_search_authors` | بحث في فهرس ٣١٠٠+ مؤلف. يعمل قبل تنزيل أي كتاب. |
| `shamela_get_page` | قراءة صفحة كاملة مع المسار من جذر فهرس الكتاب. |
| `shamela_get_toc` | فهرس كتاب على هيئة شجرة، أو سلسلة الأبواب لصفحة معيَّنة. |
| `shamela_get_book` | بيانات كتاب كاملة (المؤلف، التصنيف، النوع، السنة، الإتاحة). |
| `shamela_get_author` | بيانات مؤلف وقائمة كتبه. |
| `shamela_list_categories` | جميع التصنيفات (٤١) مع عدد الكتب لكل تصنيف. |
| `shamela_resolve` | تحويل اسم عربي جزئي إلى معرِّف كتاب أو مؤلف. |
| `shamela_get_pages_range` | قراءة نطاق متتابع من الصفحات (1–20). |
| `shamela_get_book_section` | قراءة باب كامل بمعرِّف عنوانه. |
| `shamela_get_citation` | إحالة بثلاثة أنماط (الشاملة، مختصر، موسَّع). |
| `shamela_search_quran` | بحث في القرآن الكريم. |
| `shamela_get_aya` | جلب آية بالرسم الإملائي والعثماني. |
| `shamela_get_tafseer_of_aya` | تفاسير آية بعينها. |
| `shamela_get_books_for_hadith` | كتب تروي حديثًا بعينه. |
| `shamela_list_downloaded_books` | الكتب المُنزَّلة فعليًّا على هذا الجهاز. |
| `shamela_get_book_parts` | أجزاء/مجلَّدات الكتاب. |
| `shamela_get_page_services` | إشارات الصفحة (آيات، أحاديث، أسانيد). |

## ٦) سيناريوهان نموذجيَّان

### سيناريو أول — استفسار سريع (≤ ٥ استدعاءات للأدوات)

> ما تعريف «الكلام» عند الشيخ ابن عثيمين؟

1. `shamela_resolve("ابن عثيمين")` — معرِّف المؤلف.
2. `shamela_search_pages("الكلام", scope.author_ids=[id])` — أعلى المطابقات.
3. `shamela_get_page(book_id, page_id)` — قراءة المتن.

النتيجة: الجواب من «الأصول من علم الأصول»، صفحة ١٧، مع المسار من جذر الفهرس.

### سيناريو ثانٍ — بحث موَسَّع (≤ ٥٠ استدعاءً)

> اجمع تعريف «الاستصناع» عند المذاهب الأربعة مع الإحالات.

1. `shamela_list_categories` — لمعرفة معرِّفات تصنيفات المذاهب.
2. لكل مذهب: `shamela_search_pages("استصناع", scope.category_ids=[madhhab_id])`.
3. لكل نتيجة بارزة: `shamela_get_page` ثم `shamela_get_citation` بنمط الشاملة.

النتيجة: تجميعة منظَّمة بالمذاهب مع كل إحالة جاهزة للنشر.

## ٧) أمثلة لطلبات طبيعية يفهمها كلود

- «اعرض لي الفصول التي يتحدَّث فيها ابن تيمية عن التوحيد.»
- «ابحث في القرآن عن كلمة «الصبر» وأعطني الآيات في أول سور البقرة.»
- «أيُّ كتب التفسير المنزَّلة تشرح آية {البقرة: ٢٥٥}؟»
- «أعطني الإحالة المنسَّقة لصفحة ١٧ من كتاب الأصول من علم الأصول.»
- «هل هذا الكتاب من جزء واحد أم أجزاء متعدِّدة؟ وإن كان متعدِّدًا فاعرض أسماء الأجزاء.»

## ٨) حلول للمشكلات الشائعة

**الأداة لا تظهر بعد التثبيت.**
لم تُغلِق تطبيق كلود إغلاقًا كاملًا. انقر بزر الفأرة الأيمن على الأيقونة في شريط النظام واختر «إنهاء»، ثم أعد فتح التطبيق.

**رسالة خطأ «لم يُعثر على المكتبة الشاملة».**
لم يكتشف الاكتشافُ التلقائيُّ مكانَ التثبيت. افتح إعدادات الإضافة في تطبيق كلود وأدخِل المسار يدويًّا (المجلد الذي يحتوي على المجلدين الفرعيَّين «قاعدة البيانات» و«التطبيق»).

**نتائج صفر لكلمة موجودة.**
- جرِّب البحث الصرفي بتفعيل خيار «الصرف» في طلبك (مثل: «ابحث بحثًا صرفيًّا عن…»).
- استخدم الحرفين البديلَين `*` و`?` (لا تخلط مع البحث الصرفي).
- بعض الخيارات (مراعاة التشكيل، الهمزات، الأرقام) محجوبة في الإصدار ١٫٠ وستعود في ١٫١.

**كتاب جديد منزَّل لكنه لا يظهر.**
أعد تشغيل تطبيق كلود. الإضافة تفتح الفهارس عند بداية الجلسة.

**رسالة خطأ «الكتاب غير منزَّل».**
الكتاب موجود في الفهرس لكنَّه غير منزَّل محليًّا. افتح تطبيق المكتبة الشاملة ونزِّله من هناك أوَّلًا.

## ٩) الخصوصية

كل عمليات البحث تتم على جهازك. تفتح الإضافة قاعدة بيانات المكتبة الشاملة قراءةً فقط، ولا تُرسَل أي بيانات عبر الشبكة. لا تستطيع الإضافة الكتابة في بيانات المكتبة حتى لو طُلِب منها ذلك.

## ١٠) ملاحظة على الإحالات

`shamela_get_citation` يبني الإحالة من `master.db` فقط، التي تحوي اسمَ الكتاب والمؤلف وسنة التأليف وأرقام الصفحات والأجزاء. **لا** تحوي رقم الطبعة، ولا اسم الناشر، ولا مدينة النشر، ولا اسم المحقِّق. هذه الحقول ترجع في `notes[]` كحقول مفقودة، ولا تُختلَق ولا تُلفَّق. لاستكمال الإحالة بطبعة محدَّدة، أضِفها يدويًّا أو راجِع غلاف الكتاب الورقي.

## ١١) خارطة الطريق

ميزات مخطَّطة لإصدارات لاحقة:

- مراعاة التشكيل والهمزات والأرقام.
- روابط العطف («أو») والاستثناء («وليس») بين عبارات البحث.
- البحث بالجوار والتتابع داخل العبارة.
- البحث في رواة الأحاديث.
- جلب أغلفة الكتب.
- ربط التراجم بمصادرها.

## للإبلاغ عن مشكلة

أرسل ما يظهر في سجلَّات تطبيق كلود مع وصف ما حدث، عبر صفحة المشكلات على المستودع.

## For developers

```bash
npm install                 # one-time
npm run test                # unit + integration tests (vitest)
npm run test:unit           # fast — pure functions only, no JVM
npm run test:integration    # needs a Shamela install + at least one book downloaded
npm run test:watch          # watch mode during development
npm run smoke               # legacy end-to-end smoke check
npm run pack                # build everything and produce shamela-mcp-<v>.mcpb
npm run release             # tag + publish a GitHub Release (see CLAUDE.md)
```

See [`CLAUDE.md`](./CLAUDE.md) for the testing rules, the release workflow, and [`tests/`](./tests/) for the test layout.
